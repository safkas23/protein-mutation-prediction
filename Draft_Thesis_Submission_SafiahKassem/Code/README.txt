Project Title: Protein Mutation Prediction Using Machine Learning and SHAP

Author: Safiah Kassem

Overview:
This project predicts protein mutation effects using machine learning models and protein sequence features. SHAP is used for model interpretability.

How to Run:
1. Open notebooks in Google Colab or Jupyter Notebook
2. Install dependencies:
   pip install pandas scikit-learn shap datasets
3. Run all cells from top to bottom

Outputs:
- Model performance metrics (accuracy / R² / MAE)
- SHAP feature importance plots
- Model comparison results
